Charge

The Charge entity is used to allocate permanent blockchain addresses to your customers and automatically track incoming payments to those addresses.

How it works​

  1. Create a Charge
  2. The gateway allocates one or multiple permanent addresses
  3. Your customer sends funds to these addresses
  4. The gateway detects blockchain transactions
  5. You receive webhook notifications about deposits
  6. You process payments in your system

How to create​

Request
curl -X POST "$ZERO_PAY_URL/api/charge/create" \
  -u "$ZERO_PAY_PROJECT_ID:$ZERO_PAY_PROJECT_SECRET" \
  -H "Content-Type: application/json" \
  -d '{
    "payload": "123",
    "blockchains": [
      "bitcoin",
      "litecoin",
      "tron",
      "ethereum",
      "polygon",
      "bsc",
      "arbitrum",
      "base",
      "solana",
      "ton",
      "ripple",
      "monero",
      "dogecoin"
    ]
  }'
