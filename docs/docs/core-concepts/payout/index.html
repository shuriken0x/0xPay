Payout

Payouts provide a way to programmatically send cryptocurrency payments to external wallet addresses.

A payout represents a fixed-amount outgoing transfer in a specific token and blockchain network. After creation, the gateway processes the transaction and tracks its lifecycle through webhook events.

Payouts can be used for:

  • User withdrawals
  • Balance payouts
  • Affiliate rewards
  • Automated transfers
  • Treasury operations

Each payout is processed asynchronously and produces webhook notifications reflecting the final transaction status.

Currently, payouts are supported only for EVM-compatible networks and TRON.

Payout funds and network fees are deducted from the user's primary wallet balance for the selected blockchain and token.

How to create​

Request​

Request
curl -X POST "$ZERO_PAY_URL/api/payout/create" \
  -u "$ZERO_PAY_PROJECT_ID:$ZERO_PAY_PROJECT_SECRET" \
  -H "Content-Type: application/json" \
  -d '{
  	"amount": 100,
  	"token": "USDT",
  	"blockchain": "ethereum",
  	"recipient": "0x71C7656EC7ab88b098defB751B7401B5f6d8976F",
  	"payload": "123"
  }'
FieldTypeDescription
amountnumberPayout amount in the selected token
tokenstringToken symbol used for the payout
blockchainstringBlockchain network used for the payout
recipientstringRecipient wallet address
payloadstring | nullOptional custom payload returned in webhook events and responses

Response​

Response
{
  "id": "7",
  "status": "planned",
  "payload": "123"
}
FieldTypeDescription
idstringUnique payout identifier
statusstringCurrent payout status
payloadstring | nullCustom payload passed during payout creation
StatusDescription
plannedPayout has been created and queued for processing
transaction-createdBlockchain transaction has been created
broadcastedTransaction has been broadcasted to the network
completedPayout has been successfully confirmed
failedPayout processing failed

Webhook​

Webhook Events​

EventDescription
payout:completedPayout has been successfully completed and confirmed
payout:failedPayout processing failed
