Demo Store

The Demo Store is a reference merchant application demonstrating a simple balance top-up flow using the payment gateway.

The project shows how to:

  • Create invoices for user balance top-ups
  • Handle webhook events
  • Update internal user balances after successful payments
  • Build a basic payment flow on top of the gateway primitives

The example is intentionally minimal and focuses on integration logic rather than production-ready business architecture.
